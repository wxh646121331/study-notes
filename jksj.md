# 程序员的数学基础课

## 1 二进制

- 计算机为什么要使用二进制？

  - 二进制具有抗干扰能力强、可靠性高的优点
  - 二进制非常适合做逻辑运算

- 二进制的位操作

  | 符号   | 说明                                                         |
  | ------ | ------------------------------------------------------------ |
  | <<     | 向左移位，向左移一位，就是将数据翻倍                         |
  | \>\>\> | 算术右移，算术右移保持符号位不变，除符号位之外的右移一位并补符号位1，补的1仍然在符号位之后 |
  | \>\>   | 逻辑右移，逻辑右移一位，左边补0即可                          |
